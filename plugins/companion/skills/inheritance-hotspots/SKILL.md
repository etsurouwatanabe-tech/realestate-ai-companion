# inheritance-hotspots

任意都道府県の「相続予備軍ホットスポット」を公開データで分析する。**APIキー前提・全国対応**。

## いつ使うか

- 不動産仲介会社が**自社の主力エリアの位置づけを客観的に把握したい**とき
- **複数候補エリアの優先順位**を数値で比較したいとき
- 営業エリア戦略を「勘」から「データ駆動」に切り替えたいとき
- 相続案件・任意売却・組換え提案の**ターゲット区を絞り込みたい**とき
- **都道府県をまたいで比較**したいとき（東京と神奈川の同タイプ区を探す等）

## 必要なもの

- **e-Stat APIキー**（無料・即日発行）
  - 取得先：https://www.e-stat.go.jp/api/
  - 環境変数 `ESTAT_APP_ID` に設定
- Python 3.10 以上、`uv`（推奨）

## ワークフロー（Claude Code から）

### Step 0（初回のみ）：データ準備

ユーザーが対象としたい都道府県を聞き、`/companion:prepare` を起動：

```
/companion:prepare 東京
/companion:prepare 13,14,12,11
/companion:prepare all
```

ESTAT_APP_ID 未設定の場合は取得手順を案内。所要時間は1都道府県1〜3分、全国30〜60分。

### Step 1: ユーザーの主力エリアを聞く

ユーザーが「世田谷・大田・目黒のスコアを出して」のように区を指定する。
1〜3区程度を聞き出す。指定がなければ「ご自身の主力エリアの市区町村名を教えてください」と聞く。

### Step 2: `/companion:inheritance` を実行

```
/companion:inheritance 世田谷区,大田区,目黒区
/companion:inheritance 横浜市港北区,川崎市麻生区
/companion:inheritance 東京:中央区,大阪:中央区   # 同名解決
```

### Step 3: 結果を整形して提示

各市区町村について以下を返す：

- **集計対象中の順位**
- **スコア**（高齢持ち家世帯数 × 中央値地価 / 1億）
- **5タイプ分類**（ボリューム特化／単価特化／バランス型／鉱脈型／賃貸シフト型／沈む高齢化トップ）
- **隣接区・代替候補**（同じ都道府県内で同タイプの未指定エリア）

### Step 4: 戦略示唆を1〜2行で添える

- 「主力Xエリアの平均スコア → 集計対象中Y位群 → タイプ偏在： Z型」
- 「同タイプの隣接エリアとして A区も検討余地あり」

## データソース

| データ | 出典 | 年次 |
|---|---|---|
| 高齢者持ち家世帯数 | 総務省 e-Stat 住宅・土地統計調査 | 平成30年（2018） |
| 高齢化率（65歳以上人口比率） | 総務省 e-Stat 国勢調査 | 令和2年（2020） |
| 住宅地中央値地価 | 国土交通省 国土数値情報 L01 地価公示 | 令和7年（2025）※`--year`で変更可 |
| 市区町村コード→名称 | e-Stat メタ情報 0003356490 | 自動取得・キャッシュ |

**注**：令和5年（2023年）住宅・土地統計調査の市区町村別×年齢×所有関係データはまだ未公表（2026年5月時点）。公表され次第 v1.x で更新。

## 同名市区町村の曖昧解決

「中央区」「北区」など全国に複数存在する名称は、警告＋候補一覧を返す：

```
⚠ 『中央区』は 8件の候補があります。都道府県を指定してください：
  - 北海道:中央区（札幌市）
  - 東京:中央区
  - 大阪:中央区（大阪市）
  ...
  例: --wards "東京:中央区"
```

ユーザーに「都道府県を指定してください」と促す。

## 5タイプ分類の定義

判定優先度：上から順に該当した時点で確定。

| 順 | タイプ | 判定条件 | 戦略示唆 |
|---|---|---|---|
| 1 | 沈む高齢化トップ | 持ち家率 < 50% かつ 高齢化率 ≥ 24% | 賃貸文化。相続マーケットは限定的、別商材検討 |
| 2 | 単価特化 | 中央値地価 ≥ 150万円/㎡ | 富裕層相続専門。小規模宅地・組換え提案に強み |
| 3 | ボリューム特化 | 高齢持ち家世帯 ≥ 50,000 | 面で取る。多店舗・人海戦術向き |
| 4 | 鉱脈型 | 高齢持ち家世帯 ≥ 40,000 かつ 中央値地価 < 80万円/㎡ | 業界が見落とす。先行参入で地位確保 |
| 5 | バランス型 | 持ち家率 ≥ 60% かつ 上記未該当 | 安定供給。中堅単価×中堅世帯で長期付き合い |
| 6 | 賃貸シフト型 | 持ち家率 < 60% かつ 上記未該当 | 持ち家相続案件は薄め。賃貸管理／投資物件が主戦場 |
