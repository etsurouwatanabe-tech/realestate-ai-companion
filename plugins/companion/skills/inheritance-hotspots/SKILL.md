# inheritance-hotspots

東京23区の「相続予備軍ホットスポット」を公開データで分析する。**APIキー前提のskill**。

## いつ使うか

- 不動産仲介会社が**自社の主力エリアの位置づけを客観的に把握したい**とき
- **複数候補エリアの優先順位**を数値で比較したいとき
- 営業エリア戦略を「勘」から「データ駆動」に切り替えたいとき
- 相続案件・任意売却・組換え提案の**ターゲット区を絞り込みたい**とき

## 必要なもの

- **e-Stat APIキー**（無料・即日発行）
  - 取得先：https://www.e-stat.go.jp/api/
  - 環境変数 `ESTAT_APP_ID` に設定
- Python 3.10 以上、`httpx` `dbfread` `python-dotenv`

## ワークフロー

### Step 0（初回のみ）：データ準備

```bash
# e-Stat APIキー設定
export ESTAT_APP_ID=取得したID

# 23区の住宅・人口・地価データを集計（数分）
cd plugins/companion/skills/inheritance-hotspots
python scripts/prepare.py
```

`data/tokyo23_aggregated.csv` が生成される。年次更新時は再実行するだけ。

### Step 1: ユーザーの主力エリアを聞く

ユーザーが「世田谷・大田・目黒のスコアを出して」のように区を指定する。
1〜3区程度を聞き出す。指定がなければ「ご自身の主力エリアの区名を教えてください」と聞く。

### Step 2: scripts/score.py を実行

```bash
python scripts/score.py --wards "世田谷,大田,目黒"
```

### Step 3: 結果を整形して提示

各区について以下を返す：

- **23区中の順位**
- **スコア**（高齢持ち家世帯数 × 中央値地価 / 1億）
- **5タイプ分類**（ボリューム特化／単価特化／バランス型／鉱脈型／賃貸シフト型／沈む高齢化トップ）
- **隣接区・代替候補**（補完エリアの提案）

### Step 4: 戦略示唆を1〜2行で添える

- 「主力3区の平均スコア → 23区中X位群 → タイプ偏在： Y型」
- 「同タイプの隣接区として Z区も検討余地あり」

## 出力例

```
[ 大田区 ]
  順位: 4位 / 23区中
  スコア: 289（高齢持ち家 46,880世帯 × 中央値地価 62万円/㎡）
  タイプ: 鉱脈型
  高齢化率: 22.4% / 持ち家率: 66.1%

[ 品川区 ]
  順位: 6位 / 23区中
  スコア: 276
  タイプ: バランス型
  高齢化率: 19.8% / 持ち家率: 63.3%

[ 目黒区 ]
  順位: 8位
  スコア: 223
  タイプ: バランス型
  高齢化率: 19.5%

主力3区の平均スコア: 263 → 23区中7位群
タイプ偏在: バランス型 + 鉱脈型 → ボリューム×安定の両軸戦略向き

```

## データソース

| データ | 出典 | 年次 |
|---|---|---|
| 高齢者持ち家世帯数 | 総務省 e-Stat 住宅・土地統計調査 | 平成30年（2018） |
| 高齢化率（65歳以上人口比率） | 総務省 e-Stat 国勢調査 | 令和2年（2020） |
| 住宅地中央値地価 | 国土交通省 国土数値情報 L01 地価公示 | 令和7年（2025）※`--year`で変更可 |

**注**：令和5年（2023年）住宅・土地統計調査の市区町村別×年齢×所有関係データはまだ未公表（2026年5月時点）。公表され次第 v1.0.x で `prepare.py` の参照テーブルを切り替えます。

## 5タイプ分類の定義

判定優先度：上から順に該当した時点で確定。

| 順 | タイプ | 判定条件 | 戦略示唆 |
|---|---|---|---|
| 1 | 沈む高齢化トップ | 持ち家率 < 50% かつ 高齢化率 ≥ 24% | 賃貸文化。相続マーケットは限定的、別商材検討 |
| 2 | 単価特化 | 中央値地価 ≥ 150万円/㎡ | 富裕層相続専門。小規模宅地・組換え提案に強み |
| 3 | ボリューム特化 | 高齢持ち家世帯 ≥ 50,000 | 面で取る。多店舗・人海戦術向き |
| 4 | 鉱脈型 | 高齢持ち家世帯 ≥ 40,000 かつ 中央値地価 < 80万円/㎡ | 業界が見落とす。先行参入で地位確保 |
| 5 | バランス型 | 持ち家率 ≥ 60% かつ 上記未該当 | 安定供給。中堅単価×中堅世帯で長期付き合い |
| 6 | 賃貸シフト型 | 持ち家率 < 60% かつ 上記未該当 | 持ち家相続案件は薄め。賃貸管理／投資物件が主戦場 |
